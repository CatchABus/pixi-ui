Class CheckBox

Creates a container based checkbox element

example
 new CheckBox({
    checked: false,
    style: {
        unchecked: new PixiSprite(Texture.from(`switch_off.png`)),
        checked: new PixiSprite(Texture.from(`switch_on.png`)),
    }
});
